# :star: 개요
- 프로젝트명 : HobbyPlace Project <br>

- 보고서 : 

- 기획 의도  <br>
  - 일상에서 다양한 취미를 쉽게 접할 수 있는 공간 제공<br>
  - 취미와 재능을 활용하여 수익을 창출할 수 있는 방법 제안 <br>
  - 공통의 관심사를 통해 서로를 연결해 주는 취미 여가 플랫폼 기획 <br>
